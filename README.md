# ⚙️ Installation

Clone the repository
git clone https://github.com/Newton2n/portfolio.git

Navigate to the folder

cd portfolio

Install dependencies
npm install

Run development server
npm run dev

Open

http://localhost:3000

# 🚀 Deployment

This project is deployed using **Vercel**.

1. Fork repository  
2. Import project in Vercel  
3. Connect GitHub  
4. Deploy
